他们打败了全世界，却输给了自己的国家！

Updated on Jun 24, 2017 with 0 views and 0 comments
# 体育 # 生活 # 兴趣爱好 # 吐槽

大天朝最近真是五档倒车。

从禁言、封号，到手动删评论、移除关键词，在这片神奇的土地上，还有什么是不能被操控的？

只要你想得和上面不一样，就要被孤立、被抹除，甚至连个正面站出来反驳你的人都没有，就他妈躲在后台，暗搓搓点一个，删除。

好，这很集体主义。

集体主义，就是集体的主子，一手决定或抹杀你存在的意义。

说白了，他不让你说的，你说过了也跟没说一样。

今天就算掀起10万+的血雨腥风，等你一觉醒来，哟嚯，白茫茫大地真干净。

以下为知乎用户“去由”的部分回答：

什么是英雄？

英雄就是权力的棋子。

到了低谷，需要你的时候，急巴巴地请你来临危受命；等你做出了成绩，好到不能再好的时候，给你随便扣个帽子，把你踢出去，轰隆隆收割你的功绩。

飞鸟尽，良弓藏；狡兔死，走狗烹。

你以为说成是改革就能正名了？你以为群众都眼瞎？

改革是什么？是把落后的改先进，把低下的生产力提高。

中国的乒乓球，现在可以说是如日中天，好得不能再好了，日本韩国纷纷效仿，有什么理由要自毁功力，在这个当口玩儿“改革”？

设立十几个副职，把刘队明升暗降地调过去，美其名曰组织结构扁平化。

说是扁平化，其实是中央集权。

耗费几代人心血建议起来的体系，盛世刚刚拉开帷幕，就被一个会议，毁于一旦。

真实原因，至今还不可说。

有管理的地方，必然有权力。最合理的决定，当然应该由最专业的人来做。体育团队不是500强大公司，不是只要局面稳定谁来都可以的。中央集权，空降领导，新来的人懂不懂乒乓球？谁也不知道。就算懂，会比一群世界冠军、奥运冠军组成的教练团队更懂吗？不懂的话，为什么要让外行指挥内行？

中国体育界，向来都是政治斗争、派系斗争、权力斗争，高于赛事成绩、高于项目、高于运动员个人发展和人格。

几十年过去，我们的GDP涨了35倍，却没有增加一丝一毫对个体的尊重，对英雄的尊重！

刘队二十多年教练生涯，成绩可以说是完美。尽管如此，却还要草率地“被退休”，以后谁还愿意为国家体育卖命？

等到在全世界面前，我们手里没有一个能打的，只剩窝里斗，你们就满意了？

这就是你们要的爱国主义，集体主义？

究竟是谁造成了很坏的社会影响？

舆论一边倒，你采取的行动，不是查看源头，而是粗暴地删评论，删热搜。

天天删，不怕删到手抽筋吗？

这和头痛切头，脚痛切脚，有什么区别！

你把民众当傻子骗，还指望民众爱戴你，一口一个爱国主义、集体主义，自己不觉得有悖伦常吗？

任何不尊重个人的集体主义，都是耍流氓！

且不说今天这件事牵涉到的运动员和教练，都毫无疑问是为国立功的大英雄；就是对于普通人，如果集体只会利用你、否定你、收割你的功绩还反过来骂你，这样的集体，你如何爱得动！

人性永恒的本质，是为自己争取利益。

自己的利益都得不到保障，其他任何事情都是空谈。

利他？别搞笑了！放长线钓大鱼的利他是一回事，前期投资是为了后期收回成本、赚到更多；而所谓的牺牲自己、照亮别人，根本就是扭曲人性、不可持续的，即使洗脑、强化一万遍，也不会有任何效果，对社会的发展更是有害无利！

孔融为什么让梨？孔融让梨是他的本性吗？不是！孔融让梨，是因为他被教育，这样做是好的，是会受到表扬的，受到表扬会让你拥有更高的社会名誉，从而拥有更多的资源和更高的地位。

小孩子天性自然，是天生的经济学家，明白用低成本换取高利润的道理。所以孔融才肯用一个梨，交换长期的好处。

孔融让梨的本质，是花小钱赚大钱，是极度的精明和自私！

可是，这里面有一个问题。一个五六岁的小孩子，为什么觉得大人的夸奖和赞赏，比好吃的大鸭梨更有价值？牺牲实打实的利益，博得虚名，这不是扭曲的三观，还有什么是扭曲的三观？！

把一个三观扭曲、算计利己的故事，包装成无私奉献，然后倾销给无知的你，企图让你以博得虚名为动机，心甘情愿让出自己的利益。这样做的人，居心是何等叵测！

如果这世上，人人皆为爱国、集体、利他这样的虚名而活，不惜把自己的利益交给他人，那么最终得益的是谁？

就是玩弄愚民之术，让你把梨让给他的人！

退一万步讲，就算你愚民之术得逞，大家都学会了让梨，都成了美德高于天的圣人，谁也不来吃这个梨。没吃过梨，自然也不懂得梨如何好吃，更没有动机去收获更多好吃的梨。那这个社会上，谁来从事生产？没有生产，你们还让个JB？？

所以，某些人一手愚民之术，一手开放生产，玩儿得很溜啊！

但是，一切驭民之术，都是以尊重民众为基础的。

水能载舟，亦能覆舟。

这么古朴的道理，到今天难道还有人脑袋跟不上屁股，思想水平比不上一千年前的唐太宗李世民？

别再让我们的英雄，打败了全世界，却输给自己的国家；别再让人民心寒！
